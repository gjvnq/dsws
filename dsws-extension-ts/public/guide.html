Documentation

This extension, along with the command-line (CLI) tool, is designed to allow users to share and use pages focused on closed project documentation without hosting a website or using a server.

To install the CLI tool, go to the dsws-cli directory and use:

go build

To create the file with the pages, use the CLI tool to compress and build the DSWS file from any static site directory with native URLs, using the following command:

./dsws pack [Directory_Path] [DSWS File Name]

For additional information about the cli tool, use:

./dsws help

After creating the file, you can share it with the ones that need access.

To view the pages, load the DSWS file into the designated area on the home page. After that, the page can be used normally, as if it were opened directly in the browser.
